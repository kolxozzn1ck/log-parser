# Log-Parser - Анализатор лог-файлов

## Принцип работы

Программа анализирует лог-файлы, определяет типы сообщений, подсчитывает статистику и фильтрует вывод по заданным критериям.

### Процесс обработки

1. **Чтение файла** - программа построчно читает лог-файл
2. **Определение типа** - каждая строка анализируется на наличие маркеров:
   - `[INFO]` или `INFO:` → тип INFO
   - `[WARNING]` или `WARNING:` → тип WARNING
   - `[ERROR]` или `ERROR:` → тип ERROR
   - `[DEBUG]` или `DEBUG:` → тип DEBUG
   - отсутствие маркеров → тип UNKNOWN
3. **Подсчет статистики** - накапливаются счетчики для каждого типа
4. **Фильтрация** - применяются заданные пользователем фильтры
5. **Вывод результатов** - отображаются отфильтрованные строки и итоговая статистика

## Особенности

### Определение типов сообщений

Программа распознает сообщения в следующих форматах (регистронезависимо):

| Формат | Пример |
|--------|--------|
| С квадратными скобками | `[ERROR] Connection failed` |
| С двоеточием | `WARNING: Low disk space` |
| Смешанный регистр | `[Info]`, `[Debug]`, `error:` |

### Обработка неизвестных форматов

Любое сообщение, не соответствующее известным форматам, получает тип `UNKNOWN`. Это позволяет:
- Выявлять нестандартные записи в логах
- Находить "мусорные" строки
- Анализировать логи от разных систем с разными форматами

### Фильтрация

#### По типу сообщения
- `all` - все сообщения (по умолчанию)
- `info` - только информационные
- `warning` - только предупреждения
- `error` - только ошибки
- `debug` - только отладочные
- `unknown` - только нераспознанные

#### По тексту (поиск)
- Регистронезависимый поиск
- Частичное совпадение (поиск подстроки)
- Поддержка кириллицы и латиницы

### Статистика

Программа выводит:
- Количество сообщений каждого типа
- Общее количество строк в файле
- Количество строк, прошедших фильтрацию

### Свобода регистра

Все операции сравнения выполняются без учета регистра:
- `[ERROR]`, `[error]`, `[ErRoR]` распознаются одинаково
- Поиск слова `timeout` найдет `TIMEOUT`, `Timeout`, `TIMEOUT:`


### Запуск программы
1. Загрузите папку со всеми файлами программы.
2. При необоходимости добавьте в дерикторию свой лог-файл с названием stat.log .
3. Установите Visual Studio.
4. Запустите log-parser.slnx .
